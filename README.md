# 🚖 OLA Ride Insights Dashboard

An interactive Business Intelligence Dashboard developed using **Python, MySQL, SQL, Power BI, Streamlit, Pandas, and Plotly** to analyze OLA ride booking data and generate actionable business insights.

---

# 📌 Project Workflow

This project follows a complete Data Analytics lifecycle, from raw data processing to interactive dashboard development.

---

- Collected the OLA Ride Booking dataset.
- Imported the dataset into Python using Pandas.
- Explored the dataset structure.
- Verified data types and column names.

**Tools Used**
- Python
- Pandas
